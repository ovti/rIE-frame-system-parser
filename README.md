# rIE Frame System with ETPR(k) Graph Grammar Parser

This repository contains a prototype system developed as part of a Master's thesis in Electronic Information Processing.

The goal of the project is to model a frame-based declarative knowledge system as an **interpreted graph (rIE)**.

----

- Frames are represented as graph nodes with attributes.
- Relations between frames are represented as edges.
- Provides visualisation of the frame system as a graph.
- Exports selected fragments of the system as an **rIE graph in JSON format**.
